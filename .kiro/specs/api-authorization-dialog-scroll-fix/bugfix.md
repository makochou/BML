# Bugfix Requirements Document

## Introduction

在 BML 企业管理系统的 API 账号管理模块中，点击"授权"按钮弹出的授权配置抽屉（ApiAuthorizationWorkbenchDrawer）存在布局滚动问题。当前整个抽屉界面都会出现滚动条，导致用户在浏览授权树时需要滚动整个页面，影响使用体验。

本次修复将优化滚动行为，使抽屉整体保持固定高度，仅在"授权树"卡片区域内进行滚动，从而提供更好的交互体验。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 用户在 API 账号管理页面点击"授权"按钮打开授权配置抽屉时 THEN 整个抽屉界面出现垂直滚动条

1.2 WHEN 授权树中的接口列表内容较多超出可视区域时 THEN 用户需要滚动整个抽屉页面才能查看下方的接口项

1.3 WHEN 用户向下滚动查看授权树底部内容时 THEN 抽屉顶部的筛选工具栏和侧边栏的账号画像、授权数据概览等信息会被滚动到视口外

### Expected Behavior (Correct)

2.1 WHEN 用户在 API 账号管理页面点击"授权"按钮打开授权配置抽屉时 THEN 抽屉整体应保持固定高度，不出现整体滚动条

2.2 WHEN 授权树中的接口列表内容较多超出可视区域时 THEN 用户应该只在"授权树"卡片区域内进行滚动，而不是滚动整个抽屉

2.3 WHEN 用户在授权树区域内向下滚动查看底部内容时 THEN 抽屉顶部的筛选工具栏和侧边栏的账号画像、授权数据概览等信息应保持固定可见

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 用户在授权树中勾选或取消勾选接口时 THEN 系统应继续正常更新选中状态和统计数据

3.2 WHEN 用户使用筛选工具栏（搜索、模块、方法）过滤接口时 THEN 授权树应继续正常显示过滤后的结果

3.3 WHEN 用户点击"全选当前视图"或"清空当前视图"按钮时 THEN 系统应继续正常执行批量选择操作

3.4 WHEN 用户点击"保存授权"按钮时 THEN 系统应继续正常保存授权配置并关闭抽屉

3.5 WHEN 用户展开或折叠授权树的模块和控制器节点时 THEN 树形结构应继续正常展开和折叠

3.6 WHEN 侧边栏显示账号画像和授权数据概览时 THEN 这些信息应继续正常显示且数据准确

3.7 WHEN 抽屉在不同屏幕尺寸下显示时 THEN 响应式布局应继续正常工作
