# 告警实时通知 - 前端集成指南

## 📋 问题说明

### 原有问题

在之前的实现中，告警增量轮询接口基于 `lastId`（告警ID）来判断是否有新告警。这导致以下问题：

**场景**：
1. 用户首次更新许可证，API 账号从 5 个升级到 10 个
2. 系统创建告警（ID=1）：「API 账号上限升级：从 5 变更为 10」
3. 前端轮询获取到这条告警，记录 `lastId = 1`
4. 用户再次更新许可证，API 账号从 10 个升级到 20 个
5. 系统**更新**告警（ID=1）：「API 账号上限升级：从 10 变更为 20」
6. 前端轮询时携带 `lastId = 1`，后端查询 `WHERE id > 1`，**没有返回任何数据**
7. **结果**：前端无法感知到告警已更新，用户看不到实时通知

### 根本原因

告警去重机制会**更新**现有告警而不是创建新告警，但告警的 ID 不会变化，导致基于 ID 的增量轮询无法感知到告警更新。

## ✅ 解决方案

### 核心改进

将增量轮询从**基于 ID** 改为**基于时间戳**：

- **原方案**：`GET /system/alert/latest?lastId=1`
- **新方案**：`GET /system/alert/latest?lastPollTime=1714449600000`

### 优势

1. ✅ **感知告警更新**：告警更新时 `update_time` 会变化，前端能及时感知
2. ✅ **实时通知**：用户能够收到告警更新的实时通知
3. ✅ **简化逻辑**：前端只需记录上次轮询的时间戳，无需追踪最大 ID

## 🔧 前端代码修改

### 1. 修改轮询逻辑

#### 原代码（基于 ID）

```javascript
// ❌ 旧方案：基于 lastId
let lastAlertId = null;

setInterval(async () => {
  const response = await fetch(`/system/alert/latest?lastId=${lastAlertId || ''}`);
  const result = await response.json();
  
  if (result.data && result.data.length > 0) {
    // 显示新告警
    result.data.forEach(alert => {
      showNotification(alert);
    });
    
    // 更新 lastAlertId 为最新的告警 ID
    lastAlertId = Math.max(...result.data.map(a => a.id));
  }
}, 30000); // 每 30 秒轮询一次
```

#### 新代码（基于时间戳）

```javascript
// ✅ 新方案：基于 lastPollTime
let lastPollTime = null;

setInterval(async () => {
  const response = await fetch(`/system/alert/latest?lastPollTime=${lastPollTime || ''}`);
  const result = await response.json();
  
  if (result.data && result.data.length > 0) {
    // 显示新告警
    result.data.forEach(alert => {
      showNotification(alert);
    });
    
    // 更新 lastPollTime 为当前时间戳
    lastPollTime = Date.now();
  }
}, 30000); // 每 30 秒轮询一次
```

### 2. Vue 3 完整示例

```vue
<template>
  <div class="alert-center">
    <!-- 告警图标 + 未读数量 -->
    <el-badge :value="unreadCount" :hidden="unreadCount === 0">
      <el-icon @click="showAlertPanel"><Bell /></el-icon>
    </el-badge>

    <!-- 告警面板 -->
    <el-drawer v-model="drawerVisible" title="告警中心" size="400px">
      <div class="alert-list">
        <div
          v-for="alert in alerts"
          :key="alert.id"
          :class="['alert-item', `alert-${alert.alertLevel}`]"
        >
          <div class="alert-header">
            <span class="alert-title">{{ alert.alertTitle }}</span>
            <el-tag :type="getLevelType(alert.alertLevel)" size="small">
              {{ getLevelText(alert.alertLevel) }}
            </el-tag>
          </div>
          <div class="alert-content">{{ alert.alertContent }}</div>
          <div class="alert-footer">
            <span class="alert-time">{{ alert.updateTime }}</span>
            <div class="alert-actions">
              <el-button
                v-if="alert.readStatus === 0"
                size="small"
                @click="markAsRead(alert.id)"
              >
                标记已读
              </el-button>
              <el-button size="small" @click="deleteAlert(alert.id)">
                删除
              </el-button>
            </div>
          </div>
        </div>
      </div>

      <template #footer>
        <el-button @click="markAllAsRead">全部已读</el-button>
      </template>
    </el-drawer>
  </div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue';
import { Bell } from '@element-plus/icons-vue';
import { ElMessage } from 'element-plus';

const unreadCount = ref(0);
const alerts = ref([]);
const drawerVisible = ref(false);
let lastPollTime = null; // ✅ 改为时间戳
let pollingTimer = null;

// 获取未读数量
const fetchUnreadCount = async () => {
  const response = await fetch('/system/alert/unread-count');
  const result = await response.json();
  unreadCount.value = result.data;
};

// 获取最近的告警列表
const fetchRecentAlerts = async () => {
  const response = await fetch('/system/alert/list?limit=50');
  const result = await response.json();
  alerts.value = result.data;
  // ✅ 初始化 lastPollTime 为当前时间
  lastPollTime = Date.now();
};

// 增量轮询获取最新告警
const pollLatestAlerts = async () => {
  // ✅ 使用 lastPollTime 而不是 lastId
  const response = await fetch(`/system/alert/latest?lastPollTime=${lastPollTime || ''}`);
  const result = await response.json();
  
  if (result.data && result.data.length > 0) {
    // 显示新告警通知
    result.data.forEach(alert => {
      ElMessage({
        type: getLevelType(alert.alertLevel),
        message: `${alert.alertTitle}：${alert.alertContent}`,
        duration: 5000,
        showClose: true,
      });
    });
    
    // 更新告警列表（将新告警添加到列表顶部）
    alerts.value.unshift(...result.data);
    
    // ✅ 更新 lastPollTime 为当前时间戳
    lastPollTime = Date.now();
    
    // 更新未读数量
    await fetchUnreadCount();
  }
};

// 标记单条告警为已读
const markAsRead = async (id) => {
  await fetch(`/system/alert/read/${id}`, { method: 'PUT' });
  const alert = alerts.value.find(a => a.id === id);
  if (alert) {
    alert.readStatus = 1;
  }
  await fetchUnreadCount();
};

// 一键标记所有告警为已读
const markAllAsRead = async () => {
  await fetch('/system/alert/read-all', { method: 'PUT' });
  alerts.value.forEach(alert => {
    alert.readStatus = 1;
  });
  unreadCount.value = 0;
  ElMessage.success('已全部标记为已读');
};

// 删除告警
const deleteAlert = async (id) => {
  await fetch(`/system/alert/${id}`, { method: 'DELETE' });
  alerts.value = alerts.value.filter(a => a.id !== id);
  await fetchUnreadCount();
  ElMessage.success('删除成功');
};

// 显示告警面板
const showAlertPanel = () => {
  drawerVisible.value = true;
  fetchRecentAlerts();
};

// 获取告警级别对应的 Element Plus 类型
const getLevelType = (level) => {
  const typeMap = {
    info: 'info',
    warning: 'warning',
    error: 'danger',
    critical: 'danger',
  };
  return typeMap[level] || 'info';
};

// 获取告警级别文本
const getLevelText = (level) => {
  const textMap = {
    info: '提示',
    warning: '警告',
    error: '错误',
    critical: '严重',
  };
  return textMap[level] || '提示';
};

// 组件挂载时启动轮询
onMounted(() => {
  fetchUnreadCount();
  fetchRecentAlerts();
  
  // 每 30 秒轮询一次
  pollingTimer = setInterval(pollLatestAlerts, 30000);
});

// 组件卸载时清除轮询
onUnmounted(() => {
  if (pollingTimer) {
    clearInterval(pollingTimer);
  }
});

// ✅ 页面可见性优化：页面隐藏时暂停轮询，显示时恢复
document.addEventListener('visibilitychange', () => {
  if (document.hidden) {
    // 页面隐藏时暂停轮询
    if (pollingTimer) {
      clearInterval(pollingTimer);
      pollingTimer = null;
    }
  } else {
    // 页面显示时恢复轮询
    if (!pollingTimer) {
      // 立即执行一次轮询
      pollLatestAlerts();
      // 启动定时轮询
      pollingTimer = setInterval(pollLatestAlerts, 30000);
    }
  }
});
</script>
```

### 3. React 示例

```jsx
import React, { useState, useEffect, useCallback } from 'react';
import { Badge, Drawer, Button, message } from 'antd';
import { BellOutlined } from '@ant-design/icons';

const AlertCenter = () => {
  const [unreadCount, setUnreadCount] = useState(0);
  const [alerts, setAlerts] = useState([]);
  const [drawerVisible, setDrawerVisible] = useState(false);
  const [lastPollTime, setLastPollTime] = useState(null); // ✅ 改为时间戳

  // 获取未读数量
  const fetchUnreadCount = useCallback(async () => {
    const response = await fetch('/system/alert/unread-count');
    const result = await response.json();
    setUnreadCount(result.data);
  }, []);

  // 获取最近的告警列表
  const fetchRecentAlerts = useCallback(async () => {
    const response = await fetch('/system/alert/list?limit=50');
    const result = await response.json();
    setAlerts(result.data);
    // ✅ 初始化 lastPollTime 为当前时间
    setLastPollTime(Date.now());
  }, []);

  // 增量轮询获取最新告警
  const pollLatestAlerts = useCallback(async () => {
    // ✅ 使用 lastPollTime 而不是 lastId
    const response = await fetch(`/system/alert/latest?lastPollTime=${lastPollTime || ''}`);
    const result = await response.json();
    
    if (result.data && result.data.length > 0) {
      // 显示新告警通知
      result.data.forEach(alert => {
        message[getLevelType(alert.alertLevel)]({
          content: `${alert.alertTitle}：${alert.alertContent}`,
          duration: 5,
        });
      });
      
      // 更新告警列表
      setAlerts(prev => [...result.data, ...prev]);
      
      // ✅ 更新 lastPollTime 为当前时间戳
      setLastPollTime(Date.now());
      
      // 更新未读数量
      fetchUnreadCount();
    }
  }, [lastPollTime, fetchUnreadCount]);

  // 组件挂载时启动轮询
  useEffect(() => {
    fetchUnreadCount();
    fetchRecentAlerts();
    
    // 每 30 秒轮询一次
    const timer = setInterval(pollLatestAlerts, 30000);
    
    return () => clearInterval(timer);
  }, [fetchUnreadCount, fetchRecentAlerts, pollLatestAlerts]);

  // 页面可见性优化
  useEffect(() => {
    const handleVisibilityChange = () => {
      if (!document.hidden) {
        // 页面显示时立即执行一次轮询
        pollLatestAlerts();
      }
    };
    
    document.addEventListener('visibilitychange', handleVisibilityChange);
    return () => document.removeEventListener('visibilitychange', handleVisibilityChange);
  }, [pollLatestAlerts]);

  const getLevelType = (level) => {
    const typeMap = {
      info: 'info',
      warning: 'warning',
      error: 'error',
      critical: 'error',
    };
    return typeMap[level] || 'info';
  };

  return (
    <div className="alert-center">
      <Badge count={unreadCount}>
        <BellOutlined onClick={() => setDrawerVisible(true)} />
      </Badge>
      
      <Drawer
        title="告警中心"
        placement="right"
        onClose={() => setDrawerVisible(false)}
        open={drawerVisible}
        width={400}
      >
        {/* 告警列表 */}
        {alerts.map(alert => (
          <div key={alert.id} className={`alert-item alert-${alert.alertLevel}`}>
            <div className="alert-header">
              <span className="alert-title">{alert.alertTitle}</span>
            </div>
            <div className="alert-content">{alert.alertContent}</div>
            <div className="alert-footer">
              <span className="alert-time">{alert.updateTime}</span>
            </div>
          </div>
        ))}
      </Drawer>
    </div>
  );
};

export default AlertCenter;
```

## 📊 API 变更对比

### 请求参数变更

| 项目 | 原方案 | 新方案 |
|------|--------|--------|
| 参数名 | `lastId` | `lastPollTime` |
| 参数类型 | Long（告警ID） | Long（时间戳，毫秒） |
| 参数含义 | 上次已知的最大告警 ID | 上次轮询的时间戳 |
| 示例值 | `1` | `1714449600000` |

### 响应数据变更

响应数据结构**不变**，仍然返回告警列表：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": 1,
      "alertType": "LICENSE_CHANGE",
      "alertLevel": "info",
      "alertTitle": "API 账号上限升级",
      "alertContent": "许可证更新：最大 API 账号数从 10 变更为 20。",
      "readStatus": 0,
      "createTime": "2026-04-30 10:30:00",
      "updateTime": "2026-04-30 11:00:00"
    }
  ]
}
```

**注意**：`updateTime` 字段在告警更新时会变化，这是实现实时通知的关键。

## 🔍 常见问题

### Q1：为什么要改为基于时间戳？

**A**：因为告警去重机制会更新现有告警而不是创建新告警，告警的 ID 不会变化，导致基于 ID 的增量轮询无法感知到告警更新。改为基于时间戳后，告警更新时 `update_time` 会变化，前端能及时感知。

### Q2：时间戳精度是多少？

**A**：使用毫秒级时间戳（13 位数字），例如 `1714449600000`。JavaScript 的 `Date.now()` 返回的就是毫秒级时间戳。

### Q3：如果服务器时间和客户端时间不一致怎么办？

**A**：不影响。因为 `lastPollTime` 是客户端记录的上次轮询时间，服务器只需要返回 `update_time` 大于这个时间的告警即可。即使服务器时间和客户端时间有偏差，也只会影响首次轮询，后续轮询会自动同步。

### Q4：如果用户长时间不刷新页面会怎样？

**A**：不影响。轮询会持续进行，只要有新告警或告警更新，前端都能及时感知。如果担心性能问题，可以添加页面可见性优化，页面隐藏时暂停轮询。

### Q5：如何测试告警更新是否生效？

**A**：
1. 打开浏览器开发者工具
2. 在后台更新许可证（API 账号数量变更）
3. 观察 Network 标签中的 `/system/alert/latest` 请求
4. 确认响应返回更新后的告警
5. 确认前端显示实时通知
