# BML Customization 模块

## 模块说明

`bml-customization` 是 BML 平台的定制化模块，用于存放特定客户或项目的定制功能。

## 用途

- 客户专属业务逻辑
- 项目级别的个性化配置
- 定制化 UI 组件和模板

## 状态

当前为空模块，保留结构以备项目交付时按需扩展。

## 目录结构（规划）

```
bml-customization/
├── pom.xml
├── README.md
└── src/
    └── main/
        └── java/
            └── com/bml/customization/
                ├── config/     # 定制化配置
                ├── service/    # 定制化服务
                └── controller/ # 定制化接口
```
