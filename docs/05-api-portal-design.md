## 1. 功能概述

API管理中台是面向开发者的**自服务平台**，提供API发现、在线调试、账号管理、监控统计等功能。

### 1.1 核心功能矩阵

| 功能模块 | 子功能 | 优先级 |
|---------|-------|--------|
| **API目录** | 模块化分类展示 | P0 |
| | 智能搜索筛选 | P0 |
| | API详情查看 | P0 |
| **在线调试** | 请求构造器 | P0 |
| | 参数编辑器 | P0 |
| | 响应查看器 | P0 |
| | 请求历史 | P1 |
| | 环境变量 | P1 |
| **账号管理** | 账号CRUD | P0 |
| | 权限矩阵配置 | P0 |
| | 密钥管理 | P0 |
| **监控统计** | 调用量统计 | P1 |
| | 响应时间分布 | P1 |
| | 错误率追踪 | P1 |

### 4.2 API签名认证

```javascript
// 签名算法
function generateSignature(accessKey, secretKey, timestamp, body) {
  const bodyMd5 = md5(JSON.stringify(body) || '');
  const signString = `${timestamp}\n${accessKey}\n${bodyMd5}`;
  return hmacSHA256(signString, secretKey);
}

// 请求头
```

## 5. 后端支撑

### 5.1 API自动发现

```java
@Component
public class ApiRegistryService {

    @Autowired
    private RequestMappingInfoHandlerMapping handlerMapping;

    public void registerApis() {
        Map<RequestMappingInfo, HandlerMethod> map = handlerMapping.getHandlerMethods();
        for (Map.Entry<RequestMappingInfo, HandlerMethod> entry : map.entrySet()) {
            RequestMappingInfo info = entry.getKey();
            HandlerMethod method = entry.getValue();

            // 获取@Operation注解信息
            Operation operation = method.getMethodAnnotation(Operation.class);

            // 注册到数据库
            apiRegistryMapper.insert(buildApiRegistry(info, method, operation));
        }
    }
}
```
