## 2. 第一阶段：基础框架搭建 (2-3周)

### 2.1 目标
完成项目骨架、核心框架、基础设施搭建

### 2.2 任务分解

| 序号 | 任务 | 预估 | 优先级 | 输出物 |
|------|------|------|--------|--------|
| 1.1 | Maven多模块项目初始化 | 0.5天 | P0 | 项目结构 |
| 1.2 | 父POM依赖版本统一管理 | 0.5天 | P0 | pom.xml |
| 1.3 | bml-core-common开发 | 2天 | P0 | 工具类/常量/异常 |
| 1.4 | bml-core-base开发 | 2天 | P0 | 基础Entity/Service |
| 1.5 | bml-core-framework开发 | 3天 | P0 | 统一响应/异常处理 |
| 1.6 | Spring Security配置 | 2天 | P0 | 安全配置 |
| 1.7 | JWT认证实现 | 1天 | P0 | Token处理 |
| 1.8 | ShardingSphere配置 | 1天 | P0 | 分库分表配置 |
| 1.9 | Flyway集成 | 1天 | P0 | 数据库迁移 |
| 1.10 | Redis缓存配置 | 0.5天 | P0 | 缓存配置 |
| 1.11 | 日志配置 | 0.5天 | P0 | logback配置 |
| 1.12 | License验证模块 | 2天 | P0 | 授权验证 |

### 2.3 交付物
- [x] 可运行的Maven多模块项目

- [x] 安全认证配置
- [x] 数据库连接配置
- [x] 初始化SQL脚本

### 2.4 验收标准
- 项目正常启动
- 统一响应格式生效
- 全局异常处理生效
