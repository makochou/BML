## 🎨 设计概述

本设计方案为授权治理页面的表格列表区域提供了全新的视觉体验，采用现代化的卡片化设计，提升数据可读性和用户体验。

### 设计目标

- ✨ **视觉精致**：优雅的渐变背景和立体阴影
- 🎯 **易于扫描**：清晰的字体排版和间距
- 💫 **流畅交互**：悬停效果和状态反馈
- 🏷️ **现代标签**：精美的徽章和标签设计
- 📱 **响应式**：适配各种屏幕尺寸

## ✨ 视觉特性

### 1. 表格容器

**背景渐变：**
```scss
background:
  radial-gradient(ellipse at top left, rgba(99, 102, 241, 0.06), transparent 50%),
  radial-gradient(ellipse at bottom right, rgba(59, 130, 246, 0.04), transparent 50%),
  linear-gradient(135deg, 
    rgba(255, 255, 255, 0.98) 0%, 
    rgba(248, 250, 252, 0.95) 50%,
    rgba(241, 245, 249, 0.92) 100%
  );
```

**阴影效果：**
- 内部高光：`0 0 0 1px rgba(255, 255, 255, 0.8) inset`
- 主阴影：`0 20px 50px -12px rgba(0, 0, 0, 0.1)`
- 次阴影：`0 8px 16px -8px rgba(0, 0, 0, 0.06)`

### 2. 表格头部

**样式特点：**
- 高度：48px
- 背景：渐变灰色
- 字体：13px，粗体，大写
- 颜色：#1e293b
- 边框：2px 底部边框

**悬停效果：**
- 背景加深
- 平滑过渡

### 3. 表格行

**默认状态：**
- 高度：56px
- 背景：半透明白色
- 字体：14px
- 颜色：#334155

**悬停状态：**
- 背景：渐变灰白色
- 向上移动 1px
- 添加阴影

**激活状态：**
- 背景：紫色渐变
- 紫色边框光圈
- 增强阴影

### 4. 标签设计

#### 绿色标签（成功/启用）
```scss
background: linear-gradient(135deg, 
  rgba(16, 185, 129, 0.15) 0%, 
  rgba(5, 150, 105, 0.12) 100%
);
color: #047857;
```

#### 红色标签（错误/禁用）
```scss
background: linear-gradient(135deg, 
  rgba(239, 68, 68, 0.15) 0%, 
  rgba(220, 38, 38, 0.12) 100%
);
color: #b91c1c;
```

#### 蓝色标签（信息）
```scss
background: linear-gradient(135deg, 
  rgba(59, 130, 246, 0.15) 0%, 
  rgba(37, 99, 235, 0.12) 100%
);
color: #1d4ed8;
```

### 5. 操作按钮

**主要按钮：**
```scss
background: linear-gradient(135deg, 
  rgba(99, 102, 241, 0.9) 0%, 
  rgba(139, 92, 246, 0.85) 100%
);
```

**悬停效果：**
- 渐变加深
- 向上移动 1px
- 阴影增强

## 🔧 自定义配置

### 调整表格尺寸

```scss
.api-account-page .account-table {
  --table-header-height: 52px;  /* 调整头部高度 */
  --table-row-height: 60px;     /* 调整行高 */
  --table-cell-padding: 14px 18px; /* 调整单元格内边距 */
}
```

### 自定义标签颜色

```scss
/* 自定义绿色标签 */
.api-account-page .account-table :deep(.arco-tag-green) {
  background: linear-gradient(135deg, 
    rgba(your-color-1, 0.15) 0%, 
    rgba(your-color-2, 0.12) 100%
  );
  color: your-text-color;
}
```

### 调整按钮样式

```scss
.api-account-page .table-action-btn--primary {
  background: linear-gradient(135deg, 
    your-color-1 0%, 
    your-color-2 100%
  );
}
```

## 📱 响应式设计

### 桌面端（> 1024px）

- 完整的表格布局
- 所有列正常显示
- 操作按钮横向排列

### 平板端（768px - 1024px）

```scss
@media (max-width: 1024px) {
  --table-row-height: 52px;
  /* 减小按钮尺寸 */
  .table-action-btn {
    height: 30px;
    padding: 0 12px;
  }
}
```

### 移动端（< 768px）

```scss
@media (max-width: 768px) {
  --table-row-height: 48px;
  /* 减小字体和间距 */
  .arco-table-th,
  .arco-table-td {
    padding: 8px 12px;
    font-size: 13px;
  }
}
```

## ⚡ 性能优化

### GPU 加速

```scss
.account-table :deep(.arco-table-tr) {
  will-change: transform, box-shadow;
  transform: translateZ(0);
}
```

### 滚动优化

```scss
.account-table :deep(.arco-table-body) {
  scrollbar-width: thin;
  /* 自定义滚动条样式 */
}
```

### 动画优化

```scss
.account-table :deep(.arco-table-tr) {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```
