#### 设计特点：
- 🌟 **3D玻璃态背景**：多层渐变 + 模糊效果
- 🌈 **动态彩虹装饰条**：7色渐变（Indigo → Violet → Purple → Blue → Cyan → Teal → Emerald）
- 💫 **浮动光球动画**：3个光球，不同速度循环
- ✨ **输入框微光效果**：聚焦时边框发光
- 🎯 **按钮3D效果**：悬停时上浮 + 阴影加深

#### 技术实现：
